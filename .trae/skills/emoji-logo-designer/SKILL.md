# 核心流程

1. **需求洞察**：分析用户项目的精神内核、目标受众及核心价值。
2. **动物关联**：根据项目特质选择 3 种代表性动物（如：灵敏=猫/狐狸，沉稳=大象/熊，速度=猎豹/马）。
3. **品牌构建**：
    - **图标设计**：使用动物 Emoji，并为其设计艺术化背景（如：渐变色、微阴影、圆角矩形容器）。
    - **命名关联**：基于动物形象设计富有记忆点的 `app_name`（需与动物直接或隐喻相关）。
    - **文案创作**：撰写 1-3 条核心标语（Slogan）、副标题（Subtitle）及简洁的产品描述。
4. **HTML 交付**：生成一个包含 CSS 样式的单文件 HTML，展示这三个方案。

# 关键规则

- **Emoji 艺术化**：图标容器必须包含背景修饰。示例：`<div style="background: linear-gradient(135deg, #FF9A9E 0%, #FAD0C4 100%); ...">🦊</div>`。
- **三方案并列**：默认必须输出 3 个版本，且每个版本基于不同的动物。
- **文案硬约束**：
    - 标语（Slogan）必须精炼，每条 **不得超过 15 字**。
    - 必须包含：`app_name`、`slogan` (1-3条)、`subtitle`、`description`。
- **设计标准**：
    - 页面必须现代、干净、专业。
    - 仅使用单文件 HTML，所有 CSS 必须嵌入在 `<style>` 标签内。
    - 确保在现代浏览器中显示正常，且具有响应式布局。
- **生成位置**：`.trae/`目录下

# 设计逻辑说明

在输出 HTML 后，必须提供简明扼要的设计逻辑说明（Markdown 格式）：
- **选型理由**：解释为何选择这三种动物及其与项目的精神契合点。
- **视觉逻辑**：说明配色方案和图标设计如何强化品牌价值。

# 模板参考 (assets/styles.css)
- 布局：推荐使用 Flexbox 或 CSS Grid。
- 配色：使用对比明显的文字色与柔和的背景色。
- 动效：可添加简单的 Hover 缩放效果增加高级感。
